# Security Policy

## Supported Versions

Since this is a personal project/album site, I only support the latest version currently deployed to GitHub Pages.

| Version | Supported          |
| ------- | ------------------ |
| Latest  | :white_check_mark: |
| 1.0.0   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

If you find a security issue with the guestbook or the site integration, please do not open a public issue. Instead:

1. Send an email to mentallyeepy@gmail.com or contact me via my GitHub profile.
2. I will acknowledge your report within 48 hours.
3. Once a fix is deployed, I will update the site and credit you for the find if desired.
